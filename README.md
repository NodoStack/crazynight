# 🎸 Crazy Night - Digital Experience

Sitio web interactivo diseñado para la banda **Crazy Night**. 

## ✨ Características
* **Splash Screen:** Bienvenida personalizada por única vez.
* **Music Player:** Con efectos visuales de notas musicales y un personaje de anime que hace pogo al activarse.
* **Book de Fotos:** Carrusel apilado con efecto 3D dinámico.
* **Sección de Videos:** Integración con YouTube para ver a la banda en vivo.

## 🛠️ Tecnologías
* HTML5 / CSS3 (Custom Properties y Flexbox).
* JavaScript Vanilla.
* [Swiper.js](https://swiperjs.com/) para el carrusel de fotos.
